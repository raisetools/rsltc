2.1 Mathematical characters

The display syntax of RSL uses a number of mathematical characters like ∈ and ∀ that are not available in the ASCII character set. In order for RSL to be put into text files (which are what the RAISE tool uses, and are easily portable) there are ASCII equivalents for all the special characters.
